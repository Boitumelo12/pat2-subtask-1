Morse code is a method used in telecommunication to encode text characters  as standardized sequences of two different signal durations, called dots(.) and dashes (-).
The Morse code was created  for electrical telegraph in the early 1830's by Samuel Morse and Alfred Vail.
It uses an on-off mechanism that relies entirely on rhythm and timing
